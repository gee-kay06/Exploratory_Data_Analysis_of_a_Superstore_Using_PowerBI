## Exploratory Data Analysis of a Superstore Using PowerBI
Finding out the weak areas where you can work to make more profit as a business manager at a Superstore. The dataset has 9995 entries.
#### Variables
1. Ship mode
2. Segment
3. Country
4. City
5. State
6. Postal Code
7. Region
8. Category
9. Sub-category
10. Sales
11. Quantity
12. Discount
#### Target Variable
13. Profit
